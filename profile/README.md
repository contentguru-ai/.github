# ContentGuru

**AI-powered video platform for creators and organizations.**

Transform raw footage into polished, multi-platform content with AI-assisted clipping, transcription, and publishing.

## Products

- 🎬 **Momentum** — AI clip detection and review pipeline
- 📡 **Signal Command** — Live streaming broadcast control
- 📚 **Video Library** — Cloud-native asset management on Cloudflare Stream

## Stack

Cloudflare Workers · D1 · Stream · R2 · Svelte 5 · Skeleton.dev · Bun
